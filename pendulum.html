How this pendulum solver works

Core update formulas used above

  • Time stepping (for each substep of size Δt):
    vi ← vi + g · Δt
    xi* ← xi + vi · Δt
    where g = (0, -10 m/s2) is gravity.
  • XPBD distance constraint for a link of rest length L between points 0 and 1:
    C(x) = ||x1 - x0|| - L = 0
    λ ← - C(x) / (w0 + w1 + α / Δt2)
    x0 ← x0 - w0 λ n,   x1 ← x1 + w1 λ n
    with inverse masses wi = 1/mi and n the unit vector from point 0 to 1.
  • Velocity update from corrected positions:
    vi ← (xi - xiprev) / Δt
    which automatically includes constraint and collision impulses.
  • Mechanical energy (used when enforcing energy conservation):
    E = Σi ( mi g yi + ½ mi ||vi||2 )
    and velocities are rescaled so that E after the step matches E before the step.
